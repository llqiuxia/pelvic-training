# Triage 标签

技能使用五个标准 triage 角色。此文件将这些角色映射到本仓库 issue 追踪器中实际使用的标签字符串。

| 角色（技能中使用的名称） | 本仓库使用的标签 | 含义 |
| ------------------------ | ---------------- | ---- |
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待报告人补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整描述，AI 可以直接处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会被处理 |

当技能提到某个角色时（例如"应用 AFK 就绪标签"），使用上表对应的标签字符串。

如需调整标签名称，编辑右侧列即可。
